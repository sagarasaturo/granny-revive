# Rainbow Effects (Legendary Color)

This document describes the visual effects for the Legendary Rainbow color.
Designed for mobile performance and horror atmosphere.

## Color Animation

- Smooth color cycling
- Order:
  Red → Orange → Yellow → Green → Blue → Purple → Red
- Full cycle duration: 3–4 seconds
- No hard color changes
- Infinite loop while visible

## Glow Effect

- Strong outer glow
- Glow color matches current rainbow color
- Soft pulsing (slow fade in / fade out)
- No fast flashing (eye-safe)

## Particle Effects

- Small sparkle particles
- Low spawn rate
- Particles fade out slowly
- Spawn near item edges
- Disabled on low-end devices if needed

## Sound Effect (Optional)

- Soft humming sound
- Very low volume
- Plays only when item is selected
- No looping sound by default

## Performance Rules

- Only one rainbow item may animate at a time
- Particles can be disabled in battery saver mode
- Glow quality scales with device performance

## Horror Style Notes

- Dark background recommended
- Subtle and unnatural feel
- Slow movement to increase tension
- Rainbow should feel rare, not playful

## Design Rules

- Visual prestige only
- No gameplay advantage
- Same behavior on all items
